Dossier uploads de TechSuivi
==============================

Ce dossier contient les fichiers téléchargés par les utilisateurs.

Structure recommandée :
- /documents/ : Documents généraux
- /images/ : Images et photos
- /backups/ : Sauvegardes automatiques
- /temp/ : Fichiers temporaires

Le gestionnaire de fichiers permet de :
- Explorer et naviguer dans les dossiers
- Télécharger des fichiers individuels
- Créer des sauvegardes ZIP
- Supprimer des fichiers anciens
- Voir les statistiques d'utilisation
